# FoxCM

## 基本介绍
1. 基于`Python`开发，修改更加简便
2. 使用`bootstrap`和相对UI布局，界面更加适配

## 搭建方法
1. Git项目源码
    ```bash
    git clone https://github.com/ByUsi-link/FoxCM
    ```
2. 解压压缩包
    ```bash
    unzip FoxCM_1.1.zip
    ```
3. 安装Python依赖
4. 运行Python脚本
   ```bash
   python app.py
   ```
5. 浏览器访问`http://127.0.0.1:6544`，后台访问`http://127.0.0.1:6544/admin`
